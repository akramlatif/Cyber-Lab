# Cyber Security Codespace Lab

This repo provides a ready-to-use cloud lab using GitHub Codespaces.

## Tools Included
- Nmap
- Nikto
- Python (Requests, Scapy)
- Networking tools

## How to Use
1. Open in Codespaces
2. Wait for setup to complete
3. Start practicing

## Example Commands

nmap -sV scanme.nmap.org
